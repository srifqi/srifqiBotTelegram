# srifqiBotTelegram
srifqi's Bot on Telegram

## List of commands
- `/about` - About this bot
- `/calc` - Do simple math calculation
- `/help` - Display command list
- `/nameplease` - Give random name
- `/rolldice` - Roll a or n dice
- `/tosscoin` - Toss a or n coin(s)
